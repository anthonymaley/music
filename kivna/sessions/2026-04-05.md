## Gotchas
- SourceKit IDE diagnostics fire constantly for cross-file symbol resolution (`Cannot find 'verbose' in scope`, etc.) — these are false positives. The actual `swift build` compiles clean. Don't chase these.
- ArgumentParser derives CLI flag names from property names. Renaming `@Flag var verbose` to `@Flag var verboseFlag` changed the CLI flag to `--verbose-flag`. Had to use explicit `@Flag(name: [.customShort("v"), .customLong("verbose")])` to keep `--verbose`.
- The `speakerNotFound` error originally embedded the full "not found. Available: ..." sentence as the name parameter, then `errorDescription` wrapped it again as `Speaker "..." not found.` — resulting in double-wrapping. Fixed by carrying name and available list as separate tuple fields.

## Gotchas
- macOS code signing: `cp` of Mach-O binary invalidates ad-hoc signature. Must `codesign -f --sign -` after install. Can work once then fail on next launch due to signature caching.
- renderShell clearScreen removal: removing `clearScreen` from renderShell affects ALL TUIs (playlist browser, speaker picker, volume mixer, now playing). Each TUI startup needs its own one-time clear.
- Playlist names with double quotes break AppleScript string interpolation. The fix (replacingOccurrences) must be applied everywhere a playlist name is interpolated — not just the track fetch, also play, shuffle, and track selection paths.
- SourceKit diagnostics lag behind file saves. Build succeeds even when SourceKit reports errors on recently-edited variables.

## Insights
- Dirty-region rendering is the single biggest TUI performance win. The key insight: cursor movement should NEVER trigger AppleScript. Selection is local state; playback is remote state. Keep them separate.
- "Selection is not activation" applies to playlist browser too. Auto-fetching on cursor move feels heavy. Explicit Enter/Tab for activation makes browsing feel instant.
- The user caught artificial history immediately — showing playlist tracks before current index as "Played" is dishonest when they were never played. Real history requires explicit tracking.
